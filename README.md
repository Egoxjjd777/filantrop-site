# БФ «Филантроп» — Сайт премии

Современный адаптивный сайт благотворительного фонда «Филантроп» с формой онлайн-заявки на премию.

## 🌐 Демо

Сайт доступен по адресу: [ваша-ссылка.github.io](https://ваша-ссылка.github.io)

## 📁 Структура

```
filantrop/
├── static/
│   ├── index.html          # Главная страница
│   ├── application.html    # Форма заявки на премию
│   ├── styles/
│   │   ├── main.css        # Стили главной страницы
│   │   └── application.css # Стили формы заявки
│   └── scripts/
│       ├── main.js         # Скрипты главной страницы
│       └── application.js  # Логика формы заявки
└── README.md
```

## 🚀 Быстрый старт

### Локальный запуск

```bash
# Перейдите в папку static
cd static

# Запустите локальный сервер (Python)
python3 -m http.server 8000

# Откройте в браузере
open http://localhost:8000
```

## 📤 Публикация в интернете

### Вариант 1: GitHub Pages (бесплатно)

1. Создайте репозиторий на GitHub
2. Загрузите файлы из папки `static`
3. Включите GitHub Pages в настройках репозитория
4. Сайт будет доступен по адресу `https://ваш-логин.github.io/имя-репозитория`

### Вариант 2: Netlify Drop (бесплатно)

1. Откройте https://app.netlify.com/drop
2. Перетащите папку `static` в браузер
3. Готово! Сайт доступен по ссылке

### Вариант 3: Vercel (бесплатно)

1. Зарегистрируйтесь на https://vercel.com
2. Импортируйте проект из GitHub
3. Сайт автоматически опубликуется

## 📱 Форма заявки

Форма состоит из 6 шагов:
1. Заявка (тип участника, номинация)
2. Анкета (личные данные)
3. Фотография
4. Автобиография
5. Рекомендация
6. Конкурсные материалы
