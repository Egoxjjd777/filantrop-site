### 1.2 Типы контента

| Тип контента | Поля | Примеры |
|-------------|------|---------|
| **News** | Заголовок, Дата, Текст, Изображение, Категория, Теги | Новости фонда, анонсы событий |
| **Prize Nominee** | ФИО, Год, Номинация, Город, Работа, Описание, Фото | Номинанты 2012–2025 |
| **Laureate** | ФИО, Год, Номинация, Награда, Описание, Фото | Лауреаты премии |
| **Artwork** | Название, Художник, Год, Техника, Размер, Описание, Изображения | Картины музея |
| **Team Member** | ФИО, Должность, Фото, Описание обязанностей, Email | Команда фонда |
| **Partner** | Название, Логотип, Тип, Описание, Ссылка | Партнёры фонда |
| **Service** | Название, Описание, Изображение, Детали, Цены | Услуги фонда |
| **Document** | Название, Тип, Файл, Дата, Описание | Устав, положения, буклеты |
| **Event** | Название, Дата, Время, Место, Описание, Изображение | События, фестивали |

### 4.3 Скрипт экспорта из Битрикс (пример)

```php
<?php
/**
 * Пример скрипта для экспорта новостей из Битрикс
 * Сохранить как /local/php_interface/export_news.php
 */

require($_SERVER["DOCUMENT_ROOT"] . "/bitrix/modules/main/include/prolog_before.php");

use Bitrix\Main\Loader;
use Bitrix\Iblock\Iblock;
use Bitrix\Iblock\ElementTable;

Loader::includeModule('iblock');

// ID инфоблока новостей (заменить на актуальный)
$newsIblockId = 1;

// Получаем все новости
$rsElements = CIBlockElement::GetList(
    ['SORT' => 'ASC', 'ID' => 'DESC'],
    ['IBLOCK_ID' => $newsIblockId, 'ACTIVE' => 'Y'],
    false,
    false,
    ['ID', 'NAME', 'DATE_ACTIVE_FROM', 'PREVIEW_TEXT', 'DETAIL_TEXT', 'PREVIEW_PICTURE', 'DETAIL_PICTURE']
);

$exportData = [];

while ($element = $rsElements->GetNext()) {
    // Получаем изображения
    $previewImage = CFile::GetPath($element['PREVIEW_PICTURE']);
    $detailImage = CFile::GetPath($element['DETAIL_PICTURE']);
    
    // Получаем свойства (категории, теги)
    $properties = CIBlockElement::GetProperty(
        $newsIblockId, 
        $element['ID'], 
        ['SORT' => 'ASC'],
        ['CODE' => ['CATEGORY', 'TAGS']]
    );
    
    $props = [];
    while ($prop = $properties->GetNext()) {
        $props[$prop['CODE']][] = $prop['VALUE'];
    }
    
    $exportData[] = [
        'id' => $element['ID'],
        'title' => $element['NAME'],
        'date' => $element['DATE_ACTIVE_FROM'],
        'excerpt' => $element['PREVIEW_TEXT'],
        'content' => $element['DETAIL_TEXT'],
        'preview_image' => $previewImage,
        'detail_image' => $detailImage,
        'category' => $props['CATEGORY'] ?? [],
        'tags' => $props['TAGS'] ?? [],
        'meta_title' => '', // Из SEO свойств
        'meta_description' => '', // Из SEO свойств
    ];
}

// Сохраняем в JSON
file_put_contents(
    $_SERVER['DOCUMENT_ROOT'] . '/export/news.json',
    json_encode($exportData, JSON_UNESCAPED_UNICODE | JSON_PRETTY_PRINT)
);

// Экспортируем изображения
foreach ($exportData as $news) {
    foreach ([$news['preview_image'], $news['detail_image']] as $image) {
        if ($image) {
            $sourcePath = $_SERVER['DOCUMENT_ROOT'] . $image;
            $targetPath = $_SERVER['DOCUMENT_ROOT'] . '/export/images' . basename($image);
            if (file_exists($sourcePath)) {
                copy($sourcePath, $targetPath);
            }
        }
    }
}

echo "Экспорт завершён. Всего новостей: " . count($exportData);

require($_SERVER["DOCUMENT_ROOT"] . "/bitrix/modules/main/include/epilog_after.php");
?>
```
